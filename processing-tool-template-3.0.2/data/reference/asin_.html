Name asin()
float a = PI/3;
float s = sin(a);
float as = asin(s);
// Prints "1.0471976 : 0.86602545 : 1.0471976"
println(a + " : " + s + " : " + as);
Examples
 
  float a = PI + PI/3.0;
float s = sin(a);
float as = asin(s);
// Prints "4.1887903 : -0.86602545 : -1.0471976"
println(a + " : " + s + " : " + as);
Description The inverse of sin(), returns the arc sine of a value. This function expects the values in the range of -1 to 1 and values are returned in the range -PI/2 to PI/2.
Syntax asin(value)
Parameters value float: the value whose arc sine is to be returned
Returns float
